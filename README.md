# tensorflow_churn_prediction
Bu proje, TensorFlow ve Keras kullanılarak geliştirilmiş bir müşteri kaybı (churn) tahmin modelidir. Sentetik olarak üretilen müşteri verileri (yaş, harcama, abonelik süresi, destek talebi) üzerinde derin öğrenme teknikleri uygulanarak, müşterilerin şirketten ayrılma olasılığı analiz edilmektedir
